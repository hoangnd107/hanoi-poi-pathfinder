# hanoi-poi-pathfinder
A WebGIS application for exploring Points of Interest (POI) in Hanoi and finding optimal routes

## Features

- 🔍 Location search by district/ward or distance
- 🗺️ Shortest path routing from current location
- 🏛️ Points of Interest categories: tourism, shopping, health, etc.
- 🌏 Interactive map interface focused on Hanoi city
